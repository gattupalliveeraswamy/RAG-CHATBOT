About this Project

This project turns static PDFs in Google Drive into a searchable knowledge base using a Retrieval-Augmented Generation (RAG) workflow built on n8n, Pinecone, and OpenAI. It supports two ingestion modes—bulk for existing files and incremental for newly uploaded files—and a query pipeline that retrieves the most relevant context before generating answers.

How It Works (Quick Overview)

  1. Bulk ingestion: import all existing PDFs from Google Drive and vectorize them.
  2. Incremental ingestion: auto-trigger on new or updated files and refresh vectors.
  3. Query workflow: retrieve top context from Pinecone and generate an answer with OpenAI.
